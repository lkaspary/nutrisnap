# Calorie Tracker
AI-powered calorie and macro tracker. Multi-user, syncs across devices.

## Setup
1. npm install
2. Fill in .env.local with your Anthropic and Supabase keys
3. Run supabase/schema.sql in your Supabase SQL editor
4. npm run dev → open http://localhost:3000

## Deploy
Push to GitHub → import on vercel.com → add env vars → done.
